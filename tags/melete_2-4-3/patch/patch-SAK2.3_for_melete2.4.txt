Sakai 2.3.x Patch Instructions for Melete 2.4
------------------------------------------------

1. Run the script (patch/melete24patchsak23.sh). This script merges the relevant, required 
	changes of dav, content, util, jsf folders.
2. Replace Textarea to update the FCK connector.
	Replace Textarea folder with the one in /svn/sakai/textarea/trunk using revision 31556.
	rm -rf textarea
	svn checkout -r31556 https://source.sakaiproject.org/svn/textarea/trunk textarea
3. Make sure sakai builds properly after running the above script and commands as sometimes svn merge command fails.
